# Coders.mu macOS App

This folder contains the peer-facing planning notes for a possible native macOS client for Coders.mu.

The current proposal is intentionally small: a menu bar-first app that watches the next meetup and sends native macOS notifications when meaningful changes happen.

Documents:

- [prd.md](/Users/cedricpoilly/.codex/worktrees/7712/codersmu-clients/docs/macos-app/prd.md): product intent, MVP scope, and open questions
- [architecture.md](/Users/cedricpoilly/.codex/worktrees/7712/codersmu-clients/docs/macos-app/architecture.md): technical shape for a native menu bar app
- [github-issue.md](/Users/cedricpoilly/.codex/worktrees/7712/codersmu-clients/docs/macos-app/github-issue.md): concise issue body for team discussion

Status:

- discovery and architecture draft
- no native app scaffold yet
- intended for peer feedback before implementation
