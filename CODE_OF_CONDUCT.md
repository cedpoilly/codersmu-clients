# Code of Conduct

## Our Standard

This project aims to be welcoming, technically serious, and respectful.

Please:

- be direct without being hostile
- assume good intent, then clarify disagreements with evidence
- keep critique focused on code, behavior, and decisions
- make space for contributors with different levels of experience

Unacceptable behavior includes harassment, discrimination, intimidation, personal attacks, and sustained disruption of discussion or review.

## Scope

This code of conduct applies in repository discussions, issues, pull requests, and other project spaces.

## Enforcement

If you experience or witness unacceptable behavior, open a private security report or contact the maintainer through the repository owner profile. Reports will be reviewed and handled in good faith.

Contributors who violate this code of conduct may be warned, have content removed, or be blocked from participating, depending on severity.
